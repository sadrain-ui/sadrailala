## Quick Start Guide

### Installation
```bash
cd legion-engine
pnpm install
```

### Generate a Clone
```bash
# With automatic level selection
pnpm exec tsx scripts/generate-phishing-page.ts \
  --authorized-test \
  --clone-perfect \
  https://app.uniswap.org \
  ./clones/uniswap

# With tunnel fallbacks
ENABLE_CLONE_TUNNELS=true \
pnpm exec tsx scripts/generate-phishing-page.ts \
  --authorized-test \
  --mirror \
  https://protected-site.com \
  ./clones/target
```

### Deploy the Clone
```bash
# L7 Clone with full ecosystem
cd ./clones/uniswap
docker compose up

# Access at http://localhost:8080
# Full backend ecosystem operational on port 3000
```
