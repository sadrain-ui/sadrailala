# 🚀 MASTER DASHBOARD - QUICK START

## Step 1: Install Dependencies
```bash
cd apps/master-dashboard
npm install
```

## Step 2: Run Development Server
```bash
npm run dev
```

## Step 3: Open in Browser
```
http://localhost:4000
```

## Step 4: Login
```
Email: admin@legion.com
Password: password
```

## Step 5: Navigate
- Click menu items in left sidebar
- Go to "Legion Tool" to see cloning monitoring
- Check real-time metrics, platform stats, extraction logs
