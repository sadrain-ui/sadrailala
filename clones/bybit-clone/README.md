# Legion Clone - BYBIT

## Quick Start

```bash
# Start the clone
docker compose up -d

# Check logs
docker compose logs -f

# Stop
docker compose down
```

## Troubleshooting

```bash
# Check container status
docker ps

# View nginx errors
docker compose logs

# View script events
docker compose logs | grep -i "connection|error|capture"

# Rebuild
docker compose down && docker compose up
```
